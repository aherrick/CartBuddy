# CartBuddy 🛒

A tiny Blazor app that helps you find products and build a Kroger cart.

**Check it out:** https://cartbuddy.azurewebsites.net/

---

## What it does

- Find nearby Kroger stores by ZIP
- Search products
- Build a cart and start checkout (OAuth)

---

## Built with

- Blazor WebAssembly (client)
- .NET Minimal APIs (server)
- Shared models for client/server DTOs

---

## Projects

- `CartBuddy.Client` — Blazor WebAssembly UI
- `CartBuddy.Server` — Minimal API + Kroger OAuth/cart calls
- `CartBuddy.Shared` — Shared models
